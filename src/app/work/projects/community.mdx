## Overview

**Community** is a social engagement platform built with **Next.js** and **TypeScript** that allows members to connect, share updates, create events, and build a digital presence for their groups.  
The platform combines features of a **social media feed**, **news publishing**, and **event management system**, offering an all-in-one community experience.

## Key Features

- **Member Profiles**: Users can join communities, create profiles, and interact with other members.
- **Social Feed**: Members can post updates, images, and engage with community posts through likes and comments.
- **Events and News**: Admins and members can create and share events, news, and announcements to keep the community active.
- **Gallery Section**: A dedicated gallery page where members can upload and view shared media.
- **Responsive UI**: Built with **Tailwind CSS**, ensuring a consistent and accessible user experience across devices.
- **Optimized Performance**: Leveraged **Next.js** optimizations like static generation and dynamic imports to improve loading speed and scalability.

## Technologies Used

- **Next.js**: For routing, rendering, and performance optimization.
- **TypeScript**: For type-safe and scalable frontend architecture.
- **Tailwind CSS**: For modern, responsive styling.
- **React Query**: For efficient server state management.
