## Overview

**Buy Sell Medical** is an online marketplace built with **Next.js** and **TypeScript**, designed to simplify the buying and selling of refurbished medical equipment, medicines, and healthcare products.  
The platform focuses on **performance, scalability, and intuitive UI**, ensuring smooth navigation and quick product discovery for users.

## Key Features

- **Product Listings**: Users can list and explore refurbished medical devices, medicines, and accessories with detailed specifications.
- **Next.js Optimization**: Leveraged static generation (SSG) and server-side rendering (SSR) for faster load times and SEO-friendly pages.
- **TypeScript Integration**: Improved reliability and maintainability of the codebase through strict type safety and clear interface design.
- **Responsive UI**: Built with **Tailwind CSS** for a clean, modern, and mobile-friendly layout.
- **Search and Filters**: Implemented efficient product search and category-based filtering for enhanced user experience.
- **Performance Tuning**: Used image optimization, caching, and code splitting to ensure a smooth and fast experience across all devices.

## Technologies Used

- **Next.js**: For SSR, SSG, and route-based rendering.
- **TypeScript**: For type safety and maintainable code.
- **Tailwind CSS**: For fast, responsive, and consistent styling.
- **React Query**: For data fetching and caching.

## Challenges and Learnings

Balancing fast performance with a feature-rich product catalog was a core challenge. By leveraging Next.js optimizations like **image component**, **incremental static regeneration**, and **lazy loading**, the platform achieved excellent **Core Web Vitals** scores.  
This project also reinforced the value of TypeScript in catching issues early and maintaining a clean, scalable frontend structure.

## Outcome

**Buy Sell Medical** now serves as a fast, reliable, and scalable marketplace connecting healthcare professionals and suppliers.  
It provides an efficient platform for trading refurbished equipment while maintaining high performance and accessibility across devices.
